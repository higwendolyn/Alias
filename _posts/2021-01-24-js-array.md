### Array 的构造器

Array 构造器用于创建一个新的数组，通常推荐使用对象字面量的方式创建一个数组

```javascript
// 使用 Array 构造器，可以自定义长度
var a = Array(6); // [undefined x 6]
// 使用对象字面量
var b = [];
b.length = 6; // [undefined x 6]
```
